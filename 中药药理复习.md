# 中药药理复习

## 中药药性理论的现代认识

四气：

1. 对神经中枢系统
2. 对自主神经系统
3. 对内分泌系统
4. 对基础代谢

五味：

1. 辛：扩张血管、改善微循环、发汗、解热、抗病原微生物、调整胃肠平滑肌：挥发油 、苷类、生物碱
2. 酸：止泻、止血、治疗烧伤、促进胃溃疡愈合；有机酸、鞣质
3. 甘：调节机体免疫功能、影响神经系统；糖类、蛋白质、氨基酸、苷类
4. 苦：解热、抗菌、抗炎；生物碱、苷类
5. 咸：抗肿瘤、抗炎、抗菌；碘、钠、钾、钙、镁
